8 群组命令

群组命令包含以下命令：

const (
    Cmd_CreateGroup uint8 = 0x50
    Cmd_RemoveGroup
    Cmd_SetGroup
    Cmd_GetGroups
    Cmd_GetGids
    Cmd_AddGroupMembers
    Cmd_RemoveGroupMembers
    Cmd_SetGroupMember
    Cmd_GetGroupsMembers
    Cmd_GroupChangedNotification
    Cmd_GetGroupChanged
)

其中：

  • Cmd_CreateGroup: 创建群组命令，同事可以添加初始化成员
  • Cmd_RemoveGroup: 删除群组
  • Cmd_SetGroup: 设置群组信息，名称等
  • Cmd_GetGroups: 获取多个群组的信息
  • Cmd_GetGids: 获取用户所在的所有群组的GID
  • Cmd_AddGroupMembers: 向群组添加群成员
  • Cmd_RemoveGroupMembers: 删除群组成员
  • Cmd_SetGroupMember: 设置群组成员信息，群昵称、权限等
  • Cmd_GetGroupsMembers：获取所提供GID的群组的成员
  • Cmd_GroupChangedNotification: 群组变动提醒，当群组变动（删除、群名称改变、群成员变动等）服务端会向该群所有的在线成员发送此提醒
  • Cmd_GetGroupChanged: 获取群组的变化列表
